# Punto de Venta - Parada de Flores

Aplicación web sencilla para gestionar ventas en una parada de flores.

## Funcionalidades
- Añadir productos (nombre y precio)
- Seleccionar productos para cada cliente
- Calcular el precio total
- Ingresar dinero recibido y calcular el cambio
- Interfaz intuitiva y fácil de usar

## Uso
1. Añade los productos que vendes.
2. Selecciona los productos que el cliente desea comprar.
3. El sistema calcula el total.
4. Ingresa el dinero recibido y obtén el cambio.

No guarda los datos al cerrar la página.
